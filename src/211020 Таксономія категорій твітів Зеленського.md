| Категорія    | Пояснення                        |
| ------------ | -------------------------------- |
| радість      | досягнення, початок              |

| віддача      | вручення допомоги, подарунку     |
| подяка       | за отримання, за службу          |
| розповідь    | звіт про подію                   |
| співчуття    | сум, жаль                        |
| прохання     | про щось                         |
| наказ        | розпорядження, вказівка          |
| невдоволення | зауваження, застереження, вимога |
| гнів         | обурення, незгода                |
